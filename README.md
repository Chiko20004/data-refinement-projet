Data Refinement Project – Café Sales

Objectif
Ce projet a pour objectif de réaliser un raffinement complet d’un jeu de
données brut en appliquant les principes de data quality, de nettoyage
et de transformation des données.

Description des étapes

- Exploration : compréhension du jeu de données et identification des
  problèmes de qualité.
- Nettoyage : suppression des doublons, traitement des valeurs
  manquantes et correction des incohérences.
- Transformation : standardisation des colonnes, conversion des types
  de données et recalcul des valeurs dérivées.
- Export final : production d’un fichier CSV propre et exploitable.

Technologies utilisées
- Python
- Pandas
- Jupyter Notebook
- Git et GitHub
